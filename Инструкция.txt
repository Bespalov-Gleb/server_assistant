1. Установка Python 
Перейти на официальный сайт: python.org
Скачать Python 3.10 для Windows
При установке ОБЯЗАТЕЛЬНО:
Поставить галочку "Add Python to PATH"
Выбрать "Install Now"
2. Проверка установки
Открыть командную строку (cmd) и выполнить:
python --version
pip --version
Должны показать версии Python и pip
3. Установка PyCharm 
Скачать JetBrains PyCharm Community
Установить, оставив настройки по умолчанию
При первом запуске выбрать тему оформления
4. Клонирование проекта 
# Создать директорию
mkdir Projects
cd Projects
Зарегестрироваться на GitHub
# Клонировать репозиторий
git clone https://github.com/Bespalov-Gleb/server_assistant.git
cd server_assistant
5. Создание виртуального окружения 
# В директории проекта
python -m venv venv
venv\Scripts\activate
6. Установка зависимостей
pip install -r requirements.txt
